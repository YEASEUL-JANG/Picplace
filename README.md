## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/YEASEUL-JANG/Picplace.git
   ```
2. Move to Frontend Folder
   ```sh
   cd frontend
   ```   
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run a application and vue3 page 
   ```sh
   npm run serve
   ```
